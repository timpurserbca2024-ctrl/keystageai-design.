About Keystage.ai

This page explains what you’re previewing and how the files are organised.

What this site shows

  • Public UI mockups for KS1 and KS2–KS5.
  • Lightbox on image click, “View” button to open original PNG.
  • Mobile menu with a simple dropdown.

Folder structure (published)

docs/
 ├─ index.html
 ├─ about.html
 ├─ css/style.css
 ├─ js/main.js
 └─ design/
     ├─ ks1/
     └─ ks2-ks5/

How to add a new mockup

  1. Upload the image into docs/design/ks1/ or docs/design/ks2-ks5/.
  2. Add a new .card block in docs/index.html pointing to that file.
